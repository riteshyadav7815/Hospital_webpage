## 🎨 Design Elements

### Glass Morphism
```
   ╔═══════════════════════╗
   ║ Glass Container       ║  ← Translucent background
   ║ with blur effect      ║  ← Backdrop blur 20-40px
   ║ and border            ║  ← Subtle border
   ║                       ║  ← Gradient overlay
   ╚═══════════════════════╝
```

### Color Palette
```
Primary:      #0f172a (Deep Blue)
Accent:       #00c6ff (Neon Cyan)
Secondary:    #3b82f6 (Blue)
Success:      #10b981 (Green)
Warning:      #f59e0b (Orange)
```

### Typography Hierarchy
```
Heading 1: 56px Poppins Bold       - Section titles
Heading 2: 48px Poppins Bold       - Major headings
Heading 3: 28px Poppins Bold       - Card titles
Body:      16px Inter Regular      - Main text
Small:     14px Inter Regular      - Secondary text
```

### Shadows & Glows
```
Soft Glow:    0 0 20px rgba(0, 198, 255, 0.3)
Medium Glow:  0 0 30px rgba(0, 198, 255, 0.4)
Strong Glow:  0 0 50px rgba(0, 198, 255, 0.5)
Neon:         Glow + Inset glow
```

## ✨ Advanced Features

### Magnetic Buttons
Buttons follow your cursor when you hover near them, creating an interactive magnetic effect.

### 3D Tilt Cards
Specialty cards tilt in 3D space based on mouse position, creating depth.

### Glow Effects
All interactive elements glow in cyan when hovered, creating an interactive feel.

### Smooth Scroll
Buttery smooth scrolling with Lenis, not the default browser scroll.

### Auto-Carousel
Testimonials auto-rotate every 5 seconds, but you can manually control it.

### Form Animations
Input labels float up when focused, with animated underlines.
