## 🚀 Getting Started

### 1. Install Dependencies
```bash
cd prince-hospital
npm install --legacy-peer-deps
```

### 2. Run Development Server
```bash
npm run dev
```
Visits: http://localhost:3000

### 3. Build for Production
```bash
npm run build
npm start
```
